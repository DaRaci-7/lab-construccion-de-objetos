# Reflexion

Nos pareció difícil realizar la actividad en una manera que combinara eficiencia y el aprendizaje colectivo. Esto porque originalmente dividimos las tareas para realizar el laboratorio en el tiempo estimado, pero esto llevó a que algunos miembros del equipo no entendiéramos las tareas que realizaron otros miembros. Para subsanar esto, nos juntamos posteriormente a revisar en conjunto el código en nuestro repositorio GitHub y entender en conjunto.
Además de eso, existieron desafíos dentro de las tareas, ya que nos hacían combinar distintas cosas aprendidas de maneras en que no las habíamos aplicado antes, pero por los mismo nos resultó enriquecedor para asentar de mejor manera el conocimiento construido en clases.
